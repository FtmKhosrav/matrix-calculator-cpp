# 🧮 Matrix Calculator in C++

This is a simple matrix calculator written in C++ as part of my introductory programming course.

The main goal of this project was to practice basic programming concepts and get more comfortable with working with matrices and problem-solving in code.


##  What it can do

* Input matrices A and B
* Add and subtract matrices
* Multiply matrices
* Scalar multiplication
* Calculate determinant
* Compute inverse (used for division operation)
* Display and swap matrices
* Menu-based user interaction


## What I practiced with this project

* Working with 2D arrays in C++
* Breaking problems into smaller functions
* Using nested loops
* Implementing mathematical logic in code
* Handling user input in a console application

```bash id="run3"
g++ src/main.cpp -o matrix
./matrix
```


## Limitations

* Supports matrices up to 100×100 only
* Console-based application (no GUI)
* Basic input validation
* Inverse calculation implemented manually
